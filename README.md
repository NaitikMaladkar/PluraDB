# PluraDB

- Multi-provider database management (Supabase, Neon, PlanetScale, Turso, Custom/PostgreSQL)
- Direct database management — add databases by name, project URL, keys, and connection details
- All saved data stored locally as a JSON file on device
- SQL query editor with monospace font and query history
- Schema browser with expandable table/column details
- Data viewer with paginated table grid
- Export/import configuration as JSON for backup and restore
- Dark theme with offline-first local storage
- Edit existing database connections
